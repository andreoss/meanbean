MeanBean - Automated JavaBean Testing

https://github.com/meanbeanlib/meanbean

see {@link org.meanbean.test.BeanVerifications} or {@link org.meanbean.test.BeanTester}

Examples: 

	// verify java-bean getters/setters, equals, hashCode and toString for a single bean type
	BeanVerifications.verifyBean(MyBean.class);
		
	// verify beans defined in the same package
	BeanVerifications.verifyBeansIn(MyBean.class.getPackage());
